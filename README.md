# PHP GPT Assistant

一个基于 Laravel + Vue 构建的智能对话平台，支持 GPT-3.5/GPT-4 接入，支持聊天记录、用户管理、Token 统计等功能。

## 🧩 技术栈
- Laravel + MySQL + Redis
- Vue 3 + Vite + Tailwind
- JWT 认证机制
- OpenAI GPT API 接入

## 🚀 快速开始

### 本地部署
1. 克隆项目
2. 安装依赖（PHP & JS）
3. 配置 .env 文件
4. 启动服务（使用 Docker 或 artisan serve）
